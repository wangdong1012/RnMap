## Getting Started
```
$ git clone https://github.com/wangdong1012/RnMap.git
$ cd RnMap
$ npm install
$ react-native run-android
```

## 说明
小样比较简单。
使用RN利用混合开发！
HTML部分使用的svg！
虽然技术不算是复杂，但是安卓的用户rn是可以热更新的，还是很流行的技术。
没有使用高德的地铁接口是因为局限性，自己写，比较麻烦但是交互很方便。
